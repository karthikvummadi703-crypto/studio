# **App Name**: EcoPulse

## Core Features:

- Bento KPI Dashboard: An information-rich dashboard featuring glassmorphic cards that track carbon score, monthly emissions, and carbon saved metrics.
- Intelligent Footprint Calculator: A multi-step calculation tool for estimating emissions from transportation, home energy, food choices, and lifestyle habits.
- Gemini AI Strategy Tool: Uses AI as a tool to generate a personalized carbon reduction strategy and monthly action plan based on user footprint data.
- Emission Analytics Engine: Visualizes monthly trends and emission categories using interactive charts and real-time Firestore synchronization.
- Sustainability Milestone Tracker: A gamified achievement system that records carbon reduction milestones and awards badges stored in Firestore.
- Admin Management Console: Secure interface for monitoring platform-wide growth, user activity, and managing recommendation content.
- Auth Flow & Role Guard: Secure email/password authentication system with protected routes for standard users and administrators.

## Style Guidelines:

- Primary color: Vibrant Emerald (#39F3BB), chosen to evoke high-tech bio-luminescence on a dark canvas.
- Background color: Deep Onyx Green (#0F1413), providing a high-contrast foundation for glassmorphic elements.
- Accent color: Electric Forest (#31C352), an analogous green used for success states and secondary highlights.
- Font pairing: 'Space Grotesk' (sans-serif) for technical headlines and 'Inter' (sans-serif) for readable body and numerical data.
- Monolinear, modern tech icons with slight glowing borders to match the dark-theme aesthetic.
- A structured bento-grid layout for the dashboard to organize complex sustainability metrics clearly.
- Subtle framer-motion transitions between calculator steps and glassmorphic card entrance effects.
