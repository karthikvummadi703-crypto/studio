# EcoPulse AI — Environmental Strategy Node

EcoPulse AI is a high-performance carbon footprint tracking platform and AI-driven sustainability advisor built with Next.js, Firebase, and Genkit.

## 📱 Tablet Deployment Guide (Step 3: Setup)

Since you are on a tablet, follow this cheat sheet to finish Step 3 (Environment Variables) in the Vercel browser:

### 1. Where to find your keys
Open a new browser tab for each of these:

*   **Firebase Keys (6 total):**
    1.  Go to [Firebase Console](https://console.firebase.google.com/).
    2.  Select your project.
    3.  Click the **Gear Icon (⚙️)** > **Project Settings**.
    4.  Scroll down to **"Your Apps"** > **"SDK setup and configuration"**.
    5.  Select the **"Config"** radio button. You will see values like `apiKey`, `authDomain`, etc.

*   **Gemini Key (1 total):**
    1.  Go to [Google AI Studio](https://aistudio.google.com/app/apikey).
    2.  Click **"Get API Key"** and copy the key string.

### 2. Filling out the Vercel Screen
In the Vercel **"Environment Variables"** section, enter them exactly like this:

| Key (Left Box) | Value (Right Box) |
| :--- | :--- |
| `NEXT_PUBLIC_FIREBASE_API_KEY` | (Your Firebase `apiKey`) |
| `NEXT_PUBLIC_FIREBASE_AUTH_DOMAIN` | (Your Firebase `authDomain`) |
| `NEXT_PUBLIC_FIREBASE_PROJECT_ID` | (Your Firebase `projectId`) |
| `NEXT_PUBLIC_FIREBASE_STORAGE_BUCKET` | (Your Firebase `storageBucket`) |
| `NEXT_PUBLIC_FIREBASE_MESSAGING_SENDER_ID` | (Your Firebase `messagingSenderId`) |
| `NEXT_PUBLIC_FIREBASE_APP_ID` | (Your Firebase `appId`) |
| `GOOGLE_GENAI_API_KEY` | (Your Google AI Studio Key) |

### 3. Click Deploy
Once all 7 are added, tap the big **Deploy** button. Vercel will build the app and give you a live URL in about 2 minutes.

## Features

- **Carbon Impact Audit:** Real-time transportation footprint calculation.
- **AI Advisor:** Conversational Gemini-powered sustainability consultant.
- **Strategic Insights:** Personalized reduction plans generated by AI.
- **Impact Analytics:** Visual tracking of carbon evolution over time.
- **Gamified Progress:** Earn Green Points and level up your environmental status.
